react 日历组件
==============================
1. 简介：
------------------------------
一款基于react的日历组件，贯彻组件化开发思想 ，将日期选择，月份选择，年份选择功能分为不同的组件，降低代码的耦合度，使组件易复用。

支持农历日期、节假日、节气的显示

2. 用法：
------------------------------
        npm install
        webpack
        打开  index.html文件
